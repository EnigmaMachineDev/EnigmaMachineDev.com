Cadence

Privacy Policy

Effective date: May 21, 2026 App: Cadence (Android) Developer: EnigmaMachineDev

Overview

Cadence is an offline-first Android application. It does not connect to the internet, does not collect personal information, and does not transmit any data to external servers or third parties. All data you create in Cadence stays on your device.

Information We Collect

Cadence does not collect any personal information. The app stores the following data locally on your device using Android's AsyncStorage, which is private to the app:

  • Task types and their names
  • Tasks and their content
  • Custom status labels and flag types you define
  • Task progress and completion state
  • App preferences and settings

None of this data is linked to your identity, uploaded anywhere, or shared with any third party.

How We Use Your Data

Data stored by Cadence is used solely to operate the app's features on your device:

  • Displaying your tasks organized by type
  • Tracking task status and progress
  • Generating your daily task block
  • Restoring app state between sessions

Export & Import

Cadence includes an optional export feature that saves your task data as a JSON file to a location you choose. This export is performed entirely on-device using Android's Storage Access Framework — no data passes through any server. The exported file contains only the task data you created; you control where it is saved and what you do with it.

Permissions Required

Cadence requires minimal Android permissions to operate:

  • Storage Access Framework — used for the optional import and export feature. This framework does not require a broad storage permission grant; Android manages file access through the system file picker

Cadence does not request access to your contacts, location, camera, microphone, notifications, or any sensitive system resource.

Data Storage & Security

All app data is stored in your device's private AsyncStorage — a sandboxed area not accessible to other apps. This data is removed when you uninstall Cadence. If you have Android's built-in backup enabled, your device OS may back up app data to your Google account; this is governed by Google's privacy policy, not ours.

Data Sharing

Cadence does not share any data with third parties. There are no analytics SDKs, crash reporting services, advertising networks, or external APIs integrated into the app.

Deleting Your Data

You can delete all data stored by Cadence at any time by clearing the app's data in your Android settings (Settings → Apps → Cadence → Storage → Clear Data), or by uninstalling the app.

Children's Privacy

Cadence does not knowingly collect any information from children. The app collects no personal information from any user, regardless of age.

Changes to This Policy

If this privacy policy changes, the updated version will be published at this URL with a revised effective date. Continued use of Cadence after any changes constitutes acceptance of the updated policy.

Contact

If you have questions about this privacy policy, you can reach us at: github.com/EnigmaMachineDev
